# 🚗 SandrajCars Database Management System

A simple command-line Python application using MySQL to manage car records.

## 📌 Features

* Add new car records
* Update existing records
* Delete records
* Display all records

## 🛠 Technologies Used

* Python
* MySQL

## ▶️ How to Run

1. Install MySQL
2. Install mysql-connector:

   ```
   pip install mysql-connector-python
   ```
3. Run the program:

   ```
   python main.py
   ```

## 📂 Database

* Database: `SANDRAJCARS`
* Table: `ROYALCARS`
